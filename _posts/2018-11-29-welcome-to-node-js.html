Using JavaScript as a server-side programming language.

I was struggling with the Node basic concepts (HTTP, Stream, Buffer, Module) because I first started with no background knowledge about server-side programming. That made me feel quite lost for a while because I was just writing code in Node but I was not understanding at what I was doing.

At the end I realized that I was making a mistake, which is very common when starting something new. Therefore, I decided to take a look for the core elements that involves server-side programming before getting a mess in my head.

  • Mozilla Developer Network community: https://developer.mozilla.org/en-US/docs/Learn/Server-side
{% highlight javascript %} // An HTTP server that serves the sames text file for each request it receives. var http = require('http') var fs = require('fs') var port = Number(process.argv[2]) // port provided var folder = process.argv[3]; // location of the file var server = http.createServer(function(req, res) { res.writeHead(200, {'content-type': 'text/plain'}) fs.createReadStream(folder).pipe(res) }) server.listen(port) {% endhighlight %}
