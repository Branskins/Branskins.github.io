I found out that there is posible to write Javascript code to run in the Raspberry Pi thanks to a popular robotics programming framework called Johnny-Five. Also, Johnny-Five programs can be written in many other platforms like Arduino and experiment with popular libraries such as Express.js.

For more detailed information about Johnny-Five, visit the official Github page or visit their website.

In order to make that Johnny-Five works on the Raspberry Pi, is necessary to do an extra step. Look for some information in the Wiki section of this Github repository.

Here it is where I started, I only make a simple blinking LED, the equivalent to a "Hello World".

{% highlight javascript %} const five = require('johnny-five'); const raspi = require('raspi-io'); const board = five.Board({ io: new raspi() // It tells to Johnny-Five that is working on a Raspberry Pi }); board.on('ready', () => { const red = new five.Led('P1-11'); // GPIO17 red.blink(); }); {% endhighlight %}

When everything is connected, the Raspberry is configured and the code is correctly typed, the led should turn on (ignore all the cables and the button).
